==============================================================
TC01 | 재고 충분 시 주문 접수 → 즉시 CONFIRMED
==============================================================
목적: 재고가 충분한 시료를 주문하면 승인 즉시 CONFIRMED 상태가 되는지 확인

초기 재고 (samples.json 기준)
  S005 | MEMS 압력센서 시료 | 재고 100개

--------------------------------------------------------------
STEP 1. 프로그램 실행
--------------------------------------------------------------
  명령어: ./run.sh
  기대 출력: 메인 메뉴 표시

--------------------------------------------------------------
STEP 2. 주문 접수
--------------------------------------------------------------
  메인 메뉴 → 2 (주문 관리) → 1 (주문 접수)
  입력:
    시료 ID : S005
    고객명  : 한국대학교
    수량    : 10

  기대 결과:
    - "주문 접수 완료" 메시지 출력
    - 주문 ID(UUID) 화면에 표시
    - 상태: RESERVED

--------------------------------------------------------------
STEP 3. 주문 승인
--------------------------------------------------------------
  메인 메뉴 → 2 (주문 관리) → 2 (주문 승인)
  화면에 표시된 주문 목록에서 위 주문 ID 복사하여 입력

  기대 결과:
    - "주문 승인 완료 (CONFIRMED)" 메시지 출력
    - PRODUCING 없이 바로 CONFIRMED

--------------------------------------------------------------
STEP 4. 재고 반영 확인
--------------------------------------------------------------
  메인 메뉴 → 1 (시료 관리) → 2 (시료 조회)

  기대 결과:
    - S005 재고: 90개 (100 - 10)

--------------------------------------------------------------
판정 기준
  PASS: 재고 100→90 / 주문 상태 CONFIRMED
  FAIL: 상태가 PRODUCING이거나 재고가 변경되지 않은 경우
