==============================================================
TC05 | 출고 처리 → CONFIRMED → RELEASED 전환
==============================================================
목적: CONFIRMED 상태의 주문을 출고하면 RELEASED로 최종 전환되는지 확인

--------------------------------------------------------------
사전 조건
--------------------------------------------------------------
  CONFIRMED 상태 주문이 1건 이상 존재해야 합니다.
  없으면 TC01 또는 TC04를 먼저 실행하여 CONFIRMED 주문을 만드세요.

--------------------------------------------------------------
STEP 1. 프로그램 실행
--------------------------------------------------------------
  명령어: ./run.sh

--------------------------------------------------------------
STEP 2. 모니터링으로 CONFIRMED 주문 확인
--------------------------------------------------------------
  메인 메뉴 → 3 (모니터링)

  기대 결과:
    - CONFIRMED 상태 주문 1건 이상 표시
    - 주문 ID 메모

--------------------------------------------------------------
STEP 3. 출고 처리
--------------------------------------------------------------
  메인 메뉴 → 5 (출고 처리)
  화면에 표시된 CONFIRMED 주문 목록에서 주문 ID 선택/입력

  기대 결과:
    - "출고 완료 (RELEASED)" 또는 유사 성공 메시지
    - 주문 상태 RELEASED로 전환

--------------------------------------------------------------
STEP 4. 출고 후 모니터링 확인
--------------------------------------------------------------
  메인 메뉴 → 3 (모니터링)

  기대 결과:
    - 출고한 주문이 RELEASED 항목에 표시
    - CONFIRMED 목록에서는 제거

--------------------------------------------------------------
STEP 5. 영속성 확인 (재시작 후)
--------------------------------------------------------------
  프로그램 종료 후 재시작: ./run.sh
  메인 메뉴 → 3 (모니터링)

  기대 결과:
    - RELEASED 주문이 재시작 후에도 그대로 유지
    - data/orders.json 에 status: "RELEASED" 로 저장 확인

--------------------------------------------------------------
판정 기준
  PASS: 상태 RELEASED / 재시작 후에도 영속성 유지
  FAIL: 출고 후 상태가 변경되지 않거나 재시작 시 초기화
