==============================================================
TC08 | 수율 계산 정밀도 검증 (각 시료별 올림(ceil) 확인)
==============================================================
목적: 수율 계산 공식이 각 시료에 대해 소수점 올림을 정확히 처리하는지
      손 계산과 실제 재고 변화를 비교하여 검증

공식:
  실 생산량 = ceil(부족분 / (수율 × 0.9))
  총 생산시간 = 평균 생산시간 × 실 생산량

==============================================================
케이스 A: S001 실리콘 웨이퍼 8인치
  수율: 0.92 | 평균 생산시간: 120분 | 초기 재고: 50개
--------------------------------------------------------------
  주문 수량: 60개
  부족분   : 60 - 50 = 10개
  실 생산량 = ceil(10 / (0.92 × 0.9))
            = ceil(10 / 0.828)
            = ceil(12.077...)
            = 13개
  총 생산시간 = 120 × 13 = 1,560분
  처리 후 재고 = 50 + 13 - 60 = 3개

  진행:
    1. 주문 접수: S001, 수량 60
    2. 주문 승인 → PRODUCING
    3. 생산 처리 → CONFIRMED
    4. 재고 확인 → 3개 기대

==============================================================
케이스 B: S002 실리콘 웨이퍼 12인치
  수율: 0.88 | 평균 생산시간: 180분 | 초기 재고: 30개
--------------------------------------------------------------
  주문 수량: 50개
  부족분   : 50 - 30 = 20개
  실 생산량 = ceil(20 / (0.88 × 0.9))
            = ceil(20 / 0.792)
            = ceil(25.252...)
            = 26개
  총 생산시간 = 180 × 26 = 4,680분
  처리 후 재고 = 30 + 26 - 50 = 6개

  진행:
    1. 주문 접수: S002, 수량 50
    2. 주문 승인 → PRODUCING
    3. 생산 처리 → CONFIRMED
    4. 재고 확인 → 6개 기대

==============================================================
케이스 C: S004 SiC 파워 소자 시료 (부족분이 딱 나누어지는 경우)
  수율: 0.80 | 평균 생산시간: 240분 | 초기 재고: 20개
--------------------------------------------------------------
  주문 수량: 29개
  부족분   : 29 - 20 = 9개
  실 생산량 = ceil(9 / (0.80 × 0.9))
            = ceil(9 / 0.72)
            = ceil(12.5)
            = 13개   ← 정확히 올림 처리 확인
  총 생산시간 = 240 × 13 = 3,120분
  처리 후 재고 = 20 + 13 - 29 = 4개

  진행:
    1. 주문 접수: S004, 수량 29
    2. 주문 승인 → PRODUCING
    3. 생산 처리 → CONFIRMED
    4. 재고 확인 → 4개 기대

==============================================================
판정 기준
  PASS: 모든 케이스에서 재고가 손 계산과 일치
  FAIL: 재고가 1개라도 차이가 나는 경우 (반올림/버림 오류)
